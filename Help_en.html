Version: 1.09How to use 5 Minute Break

It's use to use 5 Minute Break, just left click on the system tray icon to set the new timer or to set it off. The default timer when left clicking is 1 hour, but you have more timers available in the context menu.

If you need more help, go read the docs at project's website: http://code.google.com/p/5minutebreak/
